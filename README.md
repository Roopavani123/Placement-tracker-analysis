# Placement Tracker Dashboard

A Flask-based web application to track student placement details.

## Technologies Used
- Python (Flask)
- HTML, CSS
- CSV for data storage

## How to Run
1. Install dependencies  
   pip install -r requirements.txt
2. Run the app  
   python file.py
3. Open browser  
   http://127.0.0.1:5000/
